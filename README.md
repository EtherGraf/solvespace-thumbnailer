# Thumbnailer for solvespace .slvs files

Thunar file browser

## Requires:

* solvespace
* imagemagick
```
% apt install solvespace imagemagick
```
## Install

```
% sudo make install
```

and restart Nautilus or Thunar and then your .gcode will show up rendered in the preview.

```
% thunar -q & thunar
```
